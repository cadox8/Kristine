# Kristine

> Kristine is a simple discussion forum you can easily customize to make as unique as your community.

Welcome to the documentation portal for Kristine.

Kristine is a simple discussion forum made with NodeJS and ExpressJS.

Kristine is an open source project which can be forked on [GitHub](https://github.com/cadox8/Kristine). 
Also, this documentation is [open source](https://github.com/cadox8/Kristine/tree/docs) and you can improve them.
